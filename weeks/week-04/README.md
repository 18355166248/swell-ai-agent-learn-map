## 本周目标

> 做出一个真正可演示的文档问答系统，带引用来源。

## 核心交付

**`../../projects/02-doc-rag/`**（完整版）

功能要求：

- 上传 Markdown / TXT 文档，自动切分和向量化
- 输入问题，返回答案 + 引用来源（精确到文件名和段落）
- 有一个最简单的 Web UI（Express + 静态 HTML）

**为什么引用来源很重要**：企业内部场景里 AI 最怕瞎编，引用来源让用户能自己核实，大幅提升可信度。

## 引用来源格式设计

```json
{
  "answer": "图片上传流程是...",
  "sources": [
    {
      "file": "upload-sdk.md",
      "lines": "12-18",
      "excerpt": "调用 uploadImage 方法..."
    },
    { "file": "cdn-config.md", "lines": "34-42", "excerpt": "CDN 域名配置..." }
  ]
}
```
